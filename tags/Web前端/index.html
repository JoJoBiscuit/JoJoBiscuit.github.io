一切对于Web前端开发的疑问，本质是我们不了解Web前端开发技术。

我们，只有开了上帝之眼后，在学习技术时，才不会迷失方向。

So，我们一开始，要先从大局的角度来看，Web前端是什么，从简单到复杂，能做什么。

一、Web前端能做什么？

（一）做官网

Steam官网

丰富的颜色！

强烈的视觉效果！

让我们忍不住开始玩游戏！

小红书

标杆式的首页！

优秀的排版！

看了觉得很舒服的小清新界面！

（二）做后台管理系统

Ant Design Pro

各种图表！

各种数据！

如今几乎所有互联网平台的后台管理系统，都是在浏览器网页上进行业务操作与数据管理。

Echarts

数据3D化！

现在大数据可视化的需求越来越大！

普通的后台管理系统已经渐渐地满足不了大数据时代的需求了！

（三）做社区平台

Dribbble

超级多设计图！

现代化UI交流社区！

提升前端开发者的审美圣地！

哔哩哔哩

超级多视频！

众所周知，B站是个学习网站！

无需多说，哔哩哔哩干杯！

（四）做电脑软件

Vscode

当今最流行的免费开源代码编辑器！

竟然是前端技术开发出来的！

前端开发者都用这个！

（五）做小程序

快手短视频

微信读书
